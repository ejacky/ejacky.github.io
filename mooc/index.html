  • 树

    • 5-a-1 树是为了解决向量的动态操作（插入，删除）性能很差和列表的静态操作（搜索）性能很差，是结合了2者的优势。
    • 树可以理解为列表的列表，或是2维列表，可以成为半线性结构
    • 5-a-2 树的应用 树是按照层次关系组织数据项的一种方式
    • 5-a-6 树是种无环连通图：极小连通图， 极大无环图
    • 任一节点v与根之间存在唯一路径 path(v,r) = path(v)
    • 树是半线性的， 因为前驱是唯一的， 但后继不一定唯一；图是非线性，因为前驱和后继都不唯一
    • depth(v)
    • 长子兄弟法，能实现所有的树
    • 5-c-1 深度为k的节点， 至多2^k个； 含n个节点， 高度为h的二叉树中 h < n < 2^h+1
    • 5-c-2 真二叉树， 出度为 0 或 2
    • 5-c-3 任何一个有根有序的树， 都能用二叉树来表示， 表示方法为长子，兄弟法
